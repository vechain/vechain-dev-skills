# VeChain Dev Plugin

This plugin provides a single comprehensive skill (`vechain-dev`) for AI-assisted VeChain development, covering the full stack:

- Frontend dApps, wallet connection, VeChain Kit, dapp-kit, React Query patterns
- Solidity + Hardhat on VeChainThor, gas optimization, ABI/TypeChain
- Fee delegation (VIP-191), multi-clause transactions
- Testing with Hardhat + Thor Solo
- Smart contract security review and prevention
- VeBetterDAO, StarGate staking, governance

## Skill File Conventions

- The skill directory contains a `SKILL.md` with YAML frontmatter and a `references/` subdirectory
- The SKILL.md has the operating procedure and a progressive disclosure table
- Reference files are loaded on demand based on user query topic
- Cross-references between reference files use relative paths
